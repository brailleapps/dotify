Overview

Input Requirements

A DTBook document. Notes and tables are not allowed.

Output

A PEF-document containing one or serveral volumes.

Configuration

A number of properties can be overridden by supplying a Java xml properties file to the conversion via the optional parameter "Settings file". The file should contain one or more of the following entries:

rows
Set the number of rows, i.e. the height, of the page (integer > 0)
cols
Set the number of columns, i.e. the width, of the page (integer > 0)
inner-margin
Set the inner margin, in characters (integer >= 0)
outer-margin
Set the outer margin, in characters (integer >= 0)
rowgap
Set the row gap (integer >= 0)
splitterMax
Set the volume splitter max parameter, see below
l10nrearjacketcopy
Set the text string to be used when announcing this particular block of text
l10nimagedescription
Set the text string to be used when announcing this particular block of text
l10ncolophon
Set the text string to be used when announcing this particular block of text
l10ncaption
Set the text string to be used when announcing this particular block of text

Volume Splitter

The volume splitter used in this setup will attempt to distribute pages into volumes equally with a maximum of "max" pages per volume. However, it is possible to disallow volume breaks at certain places in the text.
